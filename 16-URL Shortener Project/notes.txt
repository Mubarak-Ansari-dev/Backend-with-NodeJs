URL Shortener:-

1. Design a URL shortener service that takes in a valid URL
   and return a shortened URL, redirecting the user to the previously provided URL.

2. Also, keep track of the total visits/clicks on the URL.

  Routes
   
   POST/URL - Generates a new short URL and returns the shortened URL
               in the format exanmple.com/random-id.

   GET/:id - Redirects the user to the original URL

   GET /URL/analytics/:id - REturns the clicks for the provided short id
